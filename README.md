ZJU-UIUC Meta 战队 RoboMaster 嵌入式程序工程，基于 ChibiOS 操作系统，使用 C 和 C++。

工程包含步兵、工程、英雄、哨兵机器人源代码，适用于 RM 2018 A 型、RM 2017 开发板，通过条件编译控制编译目标。

# 基本结构

更新日期：2019.5.31

```
Meta-Infantry
├── config           CMake 配置、OpenOCD 配置、辅助脚本等
├── dev              我们编写的源码文件
├── os               ChibiOS 源码文件
├── CMakeLists.txt   CMake 主配置文件
└── README.md        此文件
```
